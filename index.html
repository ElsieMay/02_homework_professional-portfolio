Projects

Server Side APIs - Weather Dashboard App

The motivation for this project was to build a weather dashboard that will run in the browser and feature dynamically updated HTML and CSS. The dashboard uses the OpenWeather One Call API to retrieve weather data for cities.

I learnt about using API's to retrieve information, API keys, Javascript functions, event handlers and algorithms, CSS, HTML, local storage, and much more.

Galleria di Bergamo App

In this project I worked as the lead UX and UI designer to create a Bilingual art gallery app. Galleria di Bergamo is a personal guide, allowing users to navigate the Gallery of Bergamo in one place.

Lingui Languages App

 In this project I worked as the lead UX and UI designer to create a dynamic language app concept, focussing on providing accessible learning catered to young adults.

Lisa Echeveria Succulents App

 In this project I worked as the lead UX and UI designer to update the website of Lisa Echeveria. Lisa Echeveria is a boutique plants website, specialising in rare succulents and cactus varieties.
